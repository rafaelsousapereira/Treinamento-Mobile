# **Treinamento Re-Skilling Mobile - Android**

Repositório dedicado ao programa de treinamento **Re-Skilling Mobile** com foco em desenvolvimento **Android nativo utilizando Kotlin**. O projeto abrange desde os fundamentos da linguagem Kotlin até a construção de interfaces modernas com **Jetpack Compose**, incluindo navegação, componentização e boas práticas de desenvolvimento mobile.

---

## 📂 Estrutura do Repositório

O treinamento está organizado em dois módulos principais:

### 🔗 [Nivelamento — Fundamentos de Kotlin](./Android/nivelamento)

Módulo inicial com foco nos fundamentos da linguagem Kotlin aplicados por meio de um sistema CRUD via console. Explora conceitos de **POO**, **Null Safety**, **Elvis Operator**, validações e interação com o usuário utilizando `Scanner`.

### 🔗 [Projetos Android — Jetpack Compose e Navegação](./GestorFinanceiro)

Módulo avançado com desenvolvimento de aplicações Android utilizando **Jetpack Compose**. Inclui construção de interfaces, gerenciamento de estado, navegação entre telas e padrões de navegação (Drawer, Bottom Navigation).
